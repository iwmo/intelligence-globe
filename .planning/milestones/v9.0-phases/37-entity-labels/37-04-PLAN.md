**Change B — Label visibility effect (around line 349):**
Remove the early-return guard `if (labelColl.length === 0) return;` entirely. The effect must run even when the collection is empty (it will just loop over 0 items, which is a no-op). The guard was preventing the effect from running before the async worker LOADED response, but after LOADED fires the effect deps don't change so it never re-runs — removing the guard is safe because an empty loop is harmless.

The effect after the change should look like:

```typescript

  for (let i = 0; i < labelColl.length; i++) {
    const lbl = labelColl.get(i);
    const pt = collectionRef.current?.get(i);
    lbl.show = showEntityLabels && (pt?.show ?? false);
  }

```

Do not change the deps array — `satellites.data` is already in it and will re-trigger the effect after data arrives (which is after LOADED fires and populates the collection). The combination of Change A (correct show on LOADED) and Change B (no early-return) covers both the async-LOADED path and any subsequent toggle changes.
