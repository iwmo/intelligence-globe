# Phase 17 Plan 01: Schema Migration (MIG-01 Freshness Columns) Summary

**One-liner:** Hand-written Alembic migration adding 15 freshness columns across four tables (aircraft, military_aircraft, ships, gps_jamming_cells) with TDD-driven column-presence tests, enabling all v4.0 ingest workers and API routes.

## What Was Built

A single Alembic migration file (`a4f7c2e9b1d3`) that adds all freshness tracking columns required by v4.0 MIG-01, with four updated SQLAlchemy model files and an integration test scaffold.

### Migration columns added

| Table | New columns |
|-------|-------------|
| `aircraft` | `time_position`, `geo_altitude`, `vertical_rate`, `position_source`, `fetched_at`, `last_seen_at`, `is_active` |
| `military_aircraft` | `fetched_at`, `last_seen_at`, `is_active` |
| `ships` | `last_seen_at`, `is_active` |
