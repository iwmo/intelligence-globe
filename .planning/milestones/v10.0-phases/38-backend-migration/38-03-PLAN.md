    Create backend/app/tasks/ingest_adsbiol.py implementing the following contract:

    **Constants:**
    ```python
    POLL_INTERVAL_COMMERCIAL = 15   # seconds

    TRAIL_MAX_LEN = 20
    ```

    **parse_adsbiol_aircraft(ac: dict) -> dict | None:**
    - Return None if ac.get("lat") is None or ac.get("lon") is None
    - raw_alt = ac.get("alt_baro"); alt_baro = None if raw_alt == "ground" else raw_alt
    - Return dict with keys: icao24=ac.get("hex"), callsign=(ac.get("flight") or "").strip() or None, latitude, longitude, baro_altitude=alt_baro, velocity=ac.get("gs"), true_track=ac.get("track"), vertical_rate=ac.get("baro_rate"), registration=ac.get("r"), type_code=ac.get("t"), emergency=ac.get("emergency"), nav_modes=ac.get("nav_modes"), ias=ac.get("ias"), tas=ac.get("tas"), mach=ac.get("mach"), roll=ac.get("roll")
    - NO metres-to-feet conversion (ADSB.lol is already feet)
    - emergency stores the raw string (including "none") — do not convert to None

    **get_viewport_bbox() -> str | None (async):**
    - Read Redis key "globe:viewport_bbox" using os.getenv("REDIS_URL", "redis://redis:6379/0")
    - Return None if key absent or Redis error
    - If key present: raw = await client.get("globe:viewport_bbox"); decode; split by ","
    - Validate 4 parts; remap: min_lat,min_lon,max_lat,max_lon -> box_param = f"box={min_lat},{max_lat},{min_lon},{max_lon}"
    - Return the box_param string (e.g. "box=10.0,50.0,20.0,60.0")
    - Always aclose the Redis client in a finally block

    **ingest_commercial_aircraft() -> int (async):**
    - base_url = settings.adsbio_base_url (import settings from app.config)
    - box_param = await get_viewport_bbox()
    - url = f"{base_url}/?all_with_pos" + (f"&{box_param}" if box_param else "")
    - GET the URL with httpx.AsyncClient(timeout=30.0), raise_for_status()
    - Parse ac[] array; call parse_adsbiol_aircraft() for each; collect valid_records (those not None)
    - If no valid_records: log warning, return 0
    - Open AsyncSessionLocal session
    - Pre-fetch existing trails in one SELECT: select(Aircraft.icao24, Aircraft.trail) -> trail_map dict
    - For each valid record: build updated trail (append new {lon,lat,alt,ts} point, cap at TRAIL_MAX_LEN); pg_insert(Aircraft).values(...all fields including new telemetry...).on_conflict_do_update(index_elements=["icao24"], set_={...all fields...})
    - Tombstone sweep: only when box_param is None (global fetch). If seen_icao24s is non-empty and box_param is None: sa_update(Aircraft).where(Aircraft.icao24.not_in(seen_icao24s)).values(is_active=False)
    - Single session.commit()
    - Return len(valid_records)

    **ingest_military_aircraft() -> int (async):**
    - base_url = settings.adsbio_base_url
    - box_param = await get_viewport_bbox()
    - url = f"{base_url}/?all_with_pos&filter_mil" + (f"&{box_param}" if box_param else "")
    - GET url; parse "ac" array; call parse_adsbiol_aircraft() for each record
    - Map parsed dict to MilitaryAircraft fields: hex=record["icao24"], flight=record["callsign"], aircraft_type=record["type_code"], registration=record["registration"], alt_baro=record["baro_altitude"], gs=record["velocity"], track=record["true_track"], latitude, longitude, emergency=record["emergency"], nav_modes=record["nav_modes"], ias=record["ias"], tas=record["tas"], mach=record["mach"], roll=record["roll"]
    - pg_insert(MilitaryAircraft).values(**mil_record).on_conflict_do_update(index_elements=["hex"], set_={...all fields + updated_at=func.now() + fetched_at + last_seen_at + is_active=True...})
    - Tombstone sweep: same pattern — only when box_param is None
    - Trigger GPS jamming re-aggregation on success (same as existing ingest_military.py): q.enqueue(sync_aggregate_gps_jamming) from app.tasks.ingest_gps_jamming import sync_aggregate_gps_jamming
    - Return len(valid_records)

    **sync_ingest_commercial() -> None:**
    - asyncio.run(ingest_commercial_aircraft())
    - Self-re-enqueue in POLL_INTERVAL_COMMERCIAL seconds using RQ pattern (finally block)
    - Log exception and re-raise on failure (finally still fires)

    **sync_ingest_military() -> None:**
    - asyncio.run(ingest_military_aircraft())
    - Self-re-enqueue in POLL_INTERVAL_MILITARY seconds using RQ pattern (finally block)
    - Log exception and re-raise on failure (finally still fires)
