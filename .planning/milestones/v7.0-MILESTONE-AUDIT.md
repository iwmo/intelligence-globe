### E2E Flow: Camera Pan → Filtered Response

1. `App.tsx` mounts `useViewportBbox(cesiumViewer)` ✅
2. Camera `moveEnd` fires handler → computes `ViewRectangle` in degrees ✅
3. IDL guard: `minLon > maxLon` → `setViewportBbox(null)` (fallback to global) ✅
4. Null guard: `undefined rect` → `setViewportBbox(null)` (fallback to global) ✅
5. `setViewportBbox({minLat, maxLat, minLon, maxLon})` updates store ✅
6. `effectiveBbox = replayMode === 'live' ? viewportBbox : null` in all 3 hooks ✅
7. `queryKey` includes `effectiveBbox` → React Query auto-refetches on bounds change ✅
8. `URLSearchParams` builds bbox query string when `effectiveBbox` is non-null ✅
9. Backend applies `BETWEEN` filter on indexed lat/lon columns → reduced payload ✅

**Flow status: COMPLETE**
