<task type="auto">
  <name>Task 1: Create frontend/nginx.conf and update frontend/Dockerfile production stage</name>
  <files>frontend/nginx.conf, frontend/Dockerfile</files>
  <action>
Create `frontend/nginx.conf` with the exact content below — do NOT deviate from this config:

```nginx
server {
    listen 80;
    root /usr/share/nginx/html;
    index index.html;

    location /api/ {
        proxy_pass         http://backend:8000;
        proxy_http_version 1.1;
        proxy_set_header   Host $host;
        proxy_set_header   X-Real-IP $remote_addr;
        proxy_set_header   X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header   X-Forwarded-Proto $scheme;
        proxy_read_timeout 60s;
    }

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

Then update `frontend/Dockerfile`. The `builder` stage needs `ARG VITE_CESIUM_ION_TOKEN` and `ENV VITE_CESIUM_ION_TOKEN=$VITE_CESIUM_ION_TOKEN` inserted before `RUN npm run build` so the token is baked into the compiled JS bundle. The `production` stage needs `COPY nginx.conf /etc/nginx/conf.d/default.conf` added after the dist COPY.

Final Dockerfile must be:

```dockerfile
FROM node:20-alpine AS base
WORKDIR /app
COPY package*.json ./
RUN npm ci

FROM base AS development
COPY . .
EXPOSE 3000
CMD ["npm", "run", "dev", "--", "--host", "0.0.0.0"]

FROM base AS builder
ARG VITE_CESIUM_ION_TOKEN
ENV VITE_CESIUM_ION_TOKEN=$VITE_CESIUM_ION_TOKEN
COPY . .
RUN npm run build

FROM nginx:alpine AS production
COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

Critical correctness notes:
- `nginx.conf` must be at `frontend/nginx.conf` (same directory as Dockerfile) — Docker build context is `./frontend`. A file at project root will cause "COPY failed: file not found in build context".
- `try_files $uri $uri/ /index.html` is mandatory for SPA routing — omitting it causes 404 on hard refresh.
- The `proxy_pass` directive must point to `http://backend:8000` (Docker Compose service name, not localhost).

Critical correctness notes:
