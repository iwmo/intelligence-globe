CesiumJS BillboardCollection API (from research):
```typescript
import {
  BillboardCollection,
  NearFarScalar,
  BlendOption,
  Math as CesiumMath,
  Cartesian3,
} from 'cesium';

// Init (replaces PointPrimitiveCollection):
const billboards = viewer.scene.primitives.add(
  new BillboardCollection({ blendOption: BlendOption.OPAQUE })
);

// Add billboard (replaces collection.add({ position, pixelSize, color, id })):
const bb = billboards.add({
  position: Cartesian3.fromDegrees(lon, lat, altMeters),
  image: SHIP_ICON,                                    // module-scope canvas constant — same ref for all
  width: 20,
  height: 20,
  rotation: CesiumMath.toRadians(-rot),               // heading → CCW radians
  alignedAxis: Cartesian3.ZERO,                        // screen-space rotation (not globe-north)
  id: `mmsi:${ship.mmsi}`,                             // unchanged from point primitive id scheme
  scaleByDistance: new NearFarScalar(1e4, 1.5, 5e6, 0.4),
  show: layerVisible,
});

// Position update (direct assignment — same API as PointPrimitive):
bb.position = Cartesian3.fromDegrees(lon, lat, altMeters);
bb.rotation = CesiumMath.toRadians(-rot); // update heading on refresh
bb.show = layerVisible;
```

Ship heading fallback (from research — ship.heading === 511 means unavailable):
```typescript
const rot = (ship.heading !== null && ship.heading !== 511)
  ? ship.heading
  : (ship.cog ?? 0);
billboard.rotation = CesiumMath.toRadians(-rot);
```
