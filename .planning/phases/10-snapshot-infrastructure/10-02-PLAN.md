```python
def sync_ingest_military() -> None:
    try:
        asyncio.run(ingest_military_aircraft())
    except Exception as exc:
        logger.exception("Military aircraft ingest failed: %s", exc)
        raise
    finally:
        from redis import Redis
        from rq import Queue
        conn = Redis.from_url(os.getenv("REDIS_URL", "redis://redis:6379/0"))
        q = Queue(connection=conn)
        q.enqueue_in(timedelta(seconds=POLL_INTERVAL_SECONDS), sync_ingest_military)
```

Two-session DDL+DML separation pattern (anti-pitfall #3 from research):
  Session 1: ensure_partition() — DDL (CREATE TABLE IF NOT EXISTS PARTITION OF) — commit
  Session 2: batch INSERT into parent table — commit
  Never mix DDL and DML in the same transaction.
