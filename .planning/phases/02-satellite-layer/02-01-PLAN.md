Also define:
```python
CONSTELLATION_MAP = {
    "STARLINK": "Starlink", "ONEWEB": "OneWeb", "GPS": "GPS",
    "GLONASS": "GLONASS", "GALILEO": "Galileo", "BEIDOU": "Beidou",
    "ISS": "ISS", "IRIDIUM": "Iridium", "GLOBALSTAR": "Globalstar",
}

def derive_constellation(object_name: str) -> str | None:
    upper = object_name.upper()
    for prefix, name in CONSTELLATION_MAP.items():
        if upper.startswith(prefix):
            return name
    return None
```

**backend/app/api/routes_satellites.py**

Three endpoints under prefix "/satellites" (router prefix) — the app mounts them at /api:

1. GET / — list_satellites: SELECT norad_cat_id, raw_omm FROM satellites. Return [{norad_cat_id, omm}] list. Note: the route is defined without a prefix since the router itself has prefix="/satellites". However to avoid conflict with the detail route, define as @router.get("") (empty path) or the router prefix handles it.

2. GET /freshness — tle_freshness: SELECT max(updated_at). Return {last_updated: isoformat or None}. Define BEFORE the {norad_cat_id} route so FastAPI matches "freshness" literally first.

3. GET /{norad_cat_id} — get_satellite: SELECT Satellite WHERE norad_cat_id = X. Raise HTTPException(404) if not found. Derive altitude_km and velocity_km_s from mean_motion using vis-viva:
   - mu = 398600.4418, re = 6371.0
   - n_rad_s = mean_motion * 2 * pi / 86400
   - a = (mu / n_rad_s**2) ** (1/3)
   - altitude_km = round(a - re, 1)
   - velocity_km_s = round(sqrt(mu / a), 3)
   Return: {norad_cat_id, object_name, constellation, epoch, altitude_km, velocity_km_s, inclination, eccentricity, tle_updated_at}
