**frontend/src/workers/propagation.worker.ts**

Create the Web Worker file. It handles three message types:

**LOAD_OMM** — payload: Array<{norad_cat_id: number; omm: object}>
- Call satellite.json2satrec(item.omm) for each record
- Discard records where satrec.error !== 0 (silent skip, no throw)
- Store in module-scope `satrecs` array: Array<{satrec: satellite.SatRec; norad: number}>
- Respond: self.postMessage({ type: 'LOADED', count: satrecs.length })

**PROPAGATE** — payload: {timestamp: number} (ms since epoch)
- const now = new Date(payload.timestamp)
- const gmst = satellite.gstime(now)
- Allocate Float64Array of length satrecs.length * 4
- For each satrec: propagate → guard typeof pv.position === 'boolean' → eciToEcf → multiply km→m → pack [x_m, y_m, z_m, norad_id] into buffer
- On propagation failure: pack [NaN, NaN, NaN, norad_id]
- Respond with transferable: self.postMessage({ type: 'POSITIONS', buf }, [buf.buffer])

**COMPUTE_ORBIT** — payload: {omm: object; periodSeconds: number}
- const satrec = satellite.json2satrec(payload.omm) — fresh satrec for this one satellite
- stepCount = Math.ceil(payload.periodSeconds / 60)
- Iterate i = 0 to stepCount: t = new Date(now + i * 60_000)
- For each step: propagate → eciToEcf → push [x_m, y_m, z_m] to orbitPoints array
- Also: eciToGeodetic → push [longitude_rad, latitude_rad] to groundPoints array (height not needed — ground track is clamped in rendering)
- Skip steps where propagation fails (typeof pv.position === 'boolean')
- Respond: self.postMessage({ type: 'ORBIT_RESULT', orbitPoints: Array, groundPoints: Array })

Use `import * as satellite from 'satellite.js'` at the top.

The file must use the `.ts` extension. Vite handles TypeScript Web Workers natively when using `{ type: 'module' }`.

**frontend/src/hooks/useSatellites.ts**

Create a new file. Export one hook: useSatellites().

```typescript
import { useQuery } from '@tanstack/react-query';

interface SatelliteRecord {
  norad_cat_id: number;
  omm: Record<string, unknown>;
}

export function useSatellites() {
  return useQuery<SatelliteRecord[]>({
    queryKey: ['satellites'],
    queryFn: async () => {
      const controller = new AbortController();
      const timeoutId = setTimeout(() => controller.abort(), 30_000); // 30s — large payload
      try {
        const res = await fetch('/api/satellites/', { signal: controller.signal });
        if (!res.ok) throw new Error(`Satellites fetch failed: ${res.status}`);
        return res.json() as Promise<SatelliteRecord[]>;
      } finally {
        clearTimeout(timeoutId);
      }
    },
    staleTime: 7_200_000,     // 2 hours — matches backend TLE refresh interval
    refetchInterval: 7_200_000,
    retry: 3,
    retryDelay: 5_000,
  });
}
```

Note: 30-second AbortController timeout (not 5s as in health check) because the satellite payload is ~4 MB and may take longer on slow connections.
