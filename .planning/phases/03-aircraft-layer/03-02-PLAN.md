  <files>backend/app/tasks/ingest_aircraft.py</files>
  <action>
    Create backend/app/tasks/ingest_aircraft.py implementing the async ingest function and its sync RQ wrapper.

    Structure:

    1. CONSTANTS at module top:
       - OPENSKY_TOKEN_URL = "https://auth.opensky-network.org/auth/realms/opensky-network/protocol/openid-connect/token"
       - OPENSKY_STATES_URL = "https://opensky-network.org/api/states/all"
       - POLL_INTERVAL_SECONDS = 90
       - TRAIL_MAX_LEN = 20

    2. async def fetch_opensky_token(client_id: str, client_secret: str) -> str:
       - POST to TOKEN_URL with data={"grant_type": "client_credentials", "client_id": client_id, "client_secret": client_secret}
       - httpx timeout=20.0
       - resp.raise_for_status()
       - Return resp.json()["access_token"]
       - NEVER use httpx auth= parameter — use Bearer token in headers only

    3. async def fetch_aircraft_states(token: str) -> list:
       - GET STATES_URL with headers={"Authorization": f"Bearer {token}"}
       - httpx timeout=30.0
       - If resp.status_code == 429: log X-Rate-Limit-Retry-After-Seconds header, raise RuntimeError (task will not re-run immediately — self-re-enqueue at end still fires)
       - resp.raise_for_status()
       - Return data.get("states") or []

    4. async def ingest_aircraft() -> int:
       - Read OPENSKY_CLIENT_ID and OPENSKY_CLIENT_SECRET from os.environ — raise KeyError with clear message if missing
       - Call fetch_opensky_token() to get token
       - Call fetch_aircraft_states(token)
       - Filter: skip any sv where sv[5] is None or sv[6] is None (no position)
       - Pre-fetch existing trails in one SELECT before upsert loop:
         existing = await session.execute(select(Aircraft.icao24, Aircraft.trail))
         trail_map = {row.icao24: row.trail or [] for row in existing}
       - Loop over filtered states — build trail using trail_map:
         new_point = {"lon": sv[5], "lat": sv[6], "alt": sv[7], "ts": sv[4]}
         trail = (trail_map.get(icao24) or [])[-19:] + [new_point]  # cap at 20
       - Upsert using pg_insert(Aircraft).on_conflict_do_update(index_elements=["icao24"]) — update all mutable columns including trail and updated_at=func.now()
       - session.commit()
       - Log count of records upserted and skipped
       - Return count upserted

    5. def sync_ingest_aircraft() -> None:
       - RQ sync wrapper: asyncio.run(ingest_aircraft())
       - Self-re-enqueue using the exact same pattern as sync_fetch_and_upsert_satellites:
         from redis import Redis; from rq import Queue; from datetime import timedelta
         conn = Redis.from_url(os.getenv("REDIS_URL", "redis://redis:6379/0"))
         q = Queue(connection=conn)
         q.enqueue_in(timedelta(seconds=POLL_INTERVAL_SECONDS), sync_ingest_aircraft)
       - Log "Re-enqueued aircraft ingest; next run in 90s"

    CRITICAL: Never copy the `auth=(username, password)` pattern from tutorials — OpenSky Basic Auth is dead as of March 18, 2026. Only Bearer tokens work.
